# The Next Challenge 200 Plus (האתגר הבא 200 פלוס)

**Volume ID:** `NEXT_CHALLENGE`

A DOS shareware/freeware compilation CD-ROM from the mid-1990s, preserved here for historical and archival purposes.

> **Disclaimer:** I do not own this software. The contents were originally distributed on a CD-ROM compilation and are preserved here solely for historical/archival purposes. All rights belong to their respective owners.

## Contents

The CD features **~8,400 files** across **13 game categories**, totaling ~628 MB:

| Category | Directory | Notable Games |
|----------|-----------|--------------|
| **3D** | `/3D` | Doom, Doom II, Duke Nukem 3D, Wolfenstein 3D, Spear of Destiny, Heretic, Hexen, Descent, Tomb Raider, Rise of the Triad, Strife, Dark Forces, TekWar |
| **Arcade** | `/ARCADE` | Worms, Abuse, Alien Breed, Greedy (Pac-Man clone), Jazz Jackrabbit, Beavis & Butt-head, various platformers |
| **Cards** | `/CARDS` | Card games |
| **Driving** | `/DRIVING` | The Need for Speed (demo), BC Racers, Hi-Octane, Speed Haste |
| **Fighting** | `/FIGHT` | FX Fighter |
| **Flight** | `/FLIGHT` | Evasive Action |
| **Levels** | `/LEVELS` | Abuse add-ons/mods, Rayman demo levels |
| **Quest** | `/QUEST` | Adventure/RPG games |
| **Sims** | `/SIMS` | Simulation games |
| **Sport** | `/SPORT` | Sports games |
| **Strategy** | `/STRAT` | Strategy games |
| **Think** | `/THINK` | Puzzle/thinking games |

## Technical Details

- Format: ISO 9660 with Joliet extensions
- Original source: Hebrew-market CD (title in Hebrew)
- Extracted contents only (the original `.iso` file is excluded from this repo due to GitHub's file size limits)

## Usage

These are DOS-era programs. To run them:
- Use [DOSBox](https://www.dosbox.com/) or a VM with MS-DOS / FreeDOS
- Most games are in subdirectories under the category folders (e.g., `/3D/DOOM`, `/ARCADE/WORMS`)
- The original menu launcher: `Menu.exe`
