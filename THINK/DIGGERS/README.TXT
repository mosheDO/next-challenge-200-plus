                                DIGGERS
    
Welcome to DIGGERS, a fast-paced stategy/adventure game where quick 
thinking and careful decision-making combine to challenge the serious 
gamer.  

This is a shareware version of DIGGERS, which includes just one of the 
34 game levels in the complete game.  To order the complete game, mail 
or fax the order form (ORDER.FRM) to Millennium Media Group, or call 
1-800-U-WANT-IT.

IMPORTANT:  To play DIGGERS, run the Install program (INSTALL.EXE) first.
Be sure to run Install and DIGGERS itself from DOS, not from Windows.
Install will create several sub-directories which are necessary for the
game to run correctly; then it will place the files in the correct
directories. Install will also allow you to set up your sound card.

Once you have installed the program, type DIGGERS to run it.  You will be
taken to the Controller's office, where you must pick your diggers and the
zone you will be playing in.  This shareware version only allows you to 
pick the zone of AZERG.  Then you can go to the mines.  Once you are there,
click on the right mouse button to bring up the controls for your diggers.
Have fun!
