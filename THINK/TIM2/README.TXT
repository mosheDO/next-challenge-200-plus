
THE INCREDIBLE MACHINE 2 DEMO

INTRODUCTION

	This disk contains a playable demo of The Incredible Machine 2. This 
	game will challenge you with  various puzzles that must be solved by 
	inventing a machine that will successfully complete the task you are 
	given. At your disposal will be items such as gears, conveyor belts, 
	ramps, bowling balls, blimps, cannons, cats, mice, and dozens of 
	other items you can use to create your invention. Note: No animals 
	were harmed in the making of The Incredible Machine 2.

INSTALLATION INSTRUCTIONS

	After you have downloaded and decompressed the Incredible Machine 2 
	demo files to a floppy, you must first run the INSTALL.EXE program.
        This program will let you specify sound card options for the demo
        and then install the demo on your hard drive.
        You can then run the game by typing TIM2 and pressing enter.  
        You can solve some problems by choosing to create a Easy Start disk
        from the Install program.  Then reboot the computer using this disk.

HOW TO PLAY

	This Demo contains complete instructions within the game. If you hold 
	the mouse pointer over a particular icon or selection, a help balloon 
	will pop up and tell you what that item or selection is. Clicking on 
	a question mark symbol on either the main screen or the puzzle screen 
	will bring up the game's on-line help.   

	On the Puzzle screen, in the upper right corner are 3 icons -- the 
	puzzle piece, the control panel, and a flag. Clicking the Puzzle 
	Piece icon will take you to the opening screen. Clicking the Control 
	Panel will activate the Control Panel on the right side of the screen 
	where you can access other icons, such as sound options, on-line help 
	and tutorial (This will explain all the invention pieces and how they 
	function, as well as other aspects of the game), and some other 
	options. Clicking the Flag will start an invention you've made, and 
	clicking the mouse after you have started an invention will stop it.

	The Demo is very intuitive, but if you have any problems navigating 
	the screens or menus, or you have problems using certain features, 
	click on the Question Mark to get help.

NOTE: Not all features available in the actual game will be available in the 
demo. 

 TROUBLESHOOTING:

	Should you experience any difficulties with The Incredible Machine 2, 
	consult our standard troubleshooting steps below. 

Basic Troubleshooting Techniques for The Incredible Machine 2.

1) Run CHKDSK /F or SCANDISK and correct any potential disk problems. Then 
re-install (decompress) the game and re-run the INSTALL program. You might 
even consider downloading the demo again, in case there was a problem with 
the original download.

2) Try using a boot disk with the game to help insure there are no memory 
resident programs that may cause a problem. We have boot disk instructions 
available in a file called BDALL you can download. This file also contains 
an automatic boot disk creator.

3) Sound Card Problems: If you experience any sound card problems, re-run the 
INSTALL.EXE program and try using different sound settings and drivers. Also 
make sure that any drivers necessary to your sound card's operation are 
loaded properly and that there are no DMA and IRQ conflicts within the 
system. You might even try using different DMA and IRQ settings if all else 
fails. We typically recommend using DMA 1, IRQ 5 and I/O address 220. Consult 
your sound card documentation for help with configuring your sound card's 
settings and/or drivers.
