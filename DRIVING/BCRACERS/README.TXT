
BC RACERS DEMO DISK

VER. 3/7/95

========
CONTROLS
========

KEYBOARD
========

"Q" = ACCELERATES
"A" = BRAKE

"<" = TURN LEFT
">" = TURN RIGHT

Brake (A) when turning (<,>) performs handbrake turn

"Tab" = punches right

"spacebar" = activates nitro

MOUSE
=====

MOVE MOUSE LEFT = MOVES YOUR BIKE LEFT
MOVE MOUSE RIGHT = MOVES YOUR BIKE RIGHT
LEFT MOUSE BUTTON = ACCELERATES
RIGHT MOUSE BUTTON = BRAKES
DOUBLE-CLICK LEFT MOUSE BUTTON = ACTIVATES NITRO
MINUS KEY ON NUMERIC KEYPAD = PUNCHES RIGHT

IF YOU BRAKE (RIGHT MOUSE BUTTON) WHEN TURNING (MOUSE LEFT/MOUSE RIGHT)
YOU PERFORM A HANDBRAKE TURN.

JOYSTICK
========

UP = ACCELERATES
DOWN = BRAKES
LEFT = MOVES BIKE TO LEFT
RIGHT = MOVES BIKE TO RIGHT
BUTTON 1 = PUNCHES RIGHT
BUTTON 2 = ACTIVATES NITRO

IF YOU BRAKE (JOYSTICK DOWN) WHEN TURNING (JOYSTICK RIGHT/LEFT)
YOU PERFORM A HANDBRAKE TURN.

GENERAL CONTROLS
================
P = PAUSE/UNPAUSE THE GAME

*WHEN THE GAME HAS BEEN PAUSED, THE FOLLOWING FUNCTIONS ARE AVAILABLE:

GRAPHIC DETAILS
===============

F1 = HIGH DETAIL FLOOR
F2 = LOW DETAIL FLOOR
F3 = SHORT RANGE OBJECTS
F4 = MEDIUM RANGE OBJECTS
F5 = LONG RANGE OBJECTS

PLAYER 1 VIEWS
==============
F6 = LOW VIEW
F7 = MEDIUM VIEW
F8 = HIGH VIEW

PLAYER 2 VIEWS
==============
F9 = LOW VIEW
F10 = MEDIUM VIEW
F11 = HIGH VIEW

PLAYER ONE ONLY
===============
F12 = TOGGLES BETWEEN LETTER BOX & FULL SCREEN FORMAT
