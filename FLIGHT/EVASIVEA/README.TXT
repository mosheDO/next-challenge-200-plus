EVASIVE ACTION - DEMO VERSION
*****************************

WARNING - EVASIVE ACTION requires XMS memory to load. Please 
consult your DOS manual for instructions on memory configuration.


Please follow these simple steps to load the Evasive Action 
demo to your hard disk:

1. Create a directory on your hard disk where you wish the demo
   to be installed to.

2. Log to the drive that the demo disk is located i.e. A:\

3. Type "PKUNZIP -D EACTION.ZIP C:\<Directory Name>" and press return.

4. Once the demo has unpacked itself you can play the demo by typing
   "EACTION"

   N.B. SoundBlaster owners should run a program called TSSB before 
   running the demo. This loads the TSR for sound support.


The following keys have the following actions:

FUNCTION                        JOYSTICK                KEYBOARD
"Action" selected menu option   Fire Button 1           Return
"Select" next menu option       Fire Button 2           Space
Dive                            Stick Forward           @
Climb                           Stick Back              /
Turn Left                       Stick Left              Z
Turn Right                      Stick Right             X

Cockpit View                    F1      Function Key
Change Views                    F2      Function Key
Throttle Up                     +       Plus key
Throttle Down                   -       Minus Key
Toggle locator subject          TAB     Key
Close up view of locator        V       Key
Exit game                       ESC     Key
