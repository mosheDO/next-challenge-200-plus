              Suggested Brief Program Description

Wheel of Luck 2.1c is a Windows game similar to the popular 
television "Wheel of Fortune" game. Players take turns guessing
letters to solve a phrase. Supports 1-3 players, user-editable
phrase file, user-customizable Windows 3.1 sound effects via 
.WAV files if a sound board is installed. Realistic animation.
Runs in Windows Standard or Enhanced modes.
REQUIRES VBRUN200.DLL TO RUN.
