# Speed Haste (Circuit Racer 2)

A DOS racing game by Bevel to the Max / Microïds. English floppy version.

## DOSBox Setup

This game requires specific settings to run without crashes:

1. In `dosbox.conf`, set `machine=vesa_oldvbe`
2. Run `SETUP.EXE` → select **Sound Blaster 16** → save settings
3. Run `SPEED.EXE -novbl`
