                     WHAT IS SHAREWARE?

Shareware is a beautiful concept that gives you a chance  to
"try"  Chinese Checkers and other programs before you  "buy"
them.   If you try Chinese Checkers and continue to use  it,
please register. You can register this shareware version for
$17.95,  or  if  you prefer, purchase  the  enhanced  retail
version for only $2.00 more -- $19.95.

Anyone  distributing  Chinese  Checkers  for  any  kind   of
remuneration  must  first  contact ImagiSOFT,  Inc.  at  the
address  below  for authorization.   This  authorization  is
automatically  granted  to distributors  recognized  by  the
Association of Shareware Professionals (ASP) as adhering  to
its   guidelines  for  shareware  distributors,   and   such
distributors    may   begin   offering   Chinese    Checkers
immediately; however, ImagiSOFT, Inc. must still be  advised
so  that  the distributor can be kept  up-to-date  with  the
latest version of Chinese Checkers.

If  you  are  going  to distribute  Chinese  Checkers  in  a
compressed  file, please use the name CC14.??? (.ZIP,  .ARC,
.LZH, etc).  We would appreciate if you would distribute our
original CC14.ZIP whenever possible so that new users can be
assured that they aren't receiving a program infected with a
virus.


Please include the following files in your distribution copy:

    CHINESE.EXE   main program
    CHINESE.TXT   registration info, license agreement, etc.
    CC_ORDER.TXT  order form
    CHINESE.WL    graphics file(s)
    CC_I_1.WL
    CC_I_2.WL
    CC_I_3.WL
    CC_I_4.WL
    CC_I_5.WL
    CC_I_6.WL
    CHINESE.ICO   Windows icon for Chinese Checkers
    INSTALL.EXE   Hard Disk Install file(s)
    INSTALL.FIL
    READ.ME       brief description, how to install, etc.

IMPORTANT  NOTE:  The Chinese Checkers  program  checks  for
missing or altered files.  If either of the *.TXT files  has
been  altered  or if one of the *.WL files is  missing,  the
program will exit to DOS.
