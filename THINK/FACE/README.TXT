Installation:
The FACE.EXE program and FACELIB.CLK can be copied anywhere on your
hard drive. The FACEMAKE.HLP file should be in a directory on the
MS-DOS path. Or, you can simply copy it to your Windows directory
or wherever you keep all your help files.

Select "New" from the program manager File menu and then "Program Item".
Either type in the full path to locate FACE.EXE or use the Browse
button and the file selector to find FACE.EXE.
