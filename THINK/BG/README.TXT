Installation
------------

File Locations
--------------
CTL3D.DLL  Should be located in the WINDOWS directory. Must be in the PATH.
BG.EXE	   Any directory.  Must be located with BG.HLP.
BG.HLP     Any directory.  Must be located with BG.EXE.



Creating the icon
-----------------
If you already know how to install a Windows application, then place the 
executable file (BG.EXE) wherever you like.  Otherwise, copy the 
program file into your Windows directory and run Windows.  Then, 
follow these steps:

	1. In the Program Manager, open your Games icon ( if it is not 
	   already open ).

	2. Select the File menu in the menu bar of the Program 
	   Manager and make the New... selection

	3. When the dialog box opens, press the OK button.

	4. If you are using Windows 3.1, at this point you need to
	   press the BROWSE button to open the file selector.
	   Use the controls in this window to display the directory
	   that contains the files in the BattleGrid archive.

	5. In the file selector, scroll down to the bottom of the list.  
	   Double click on BG.EXE.

	6. In the next dialog that appears, press Change Icon.  If the 
	   icon shown is not acceptable, press the View Next button until
	   a more appropriate icon is displayed.  When you are ready,
	   press OK and OK again.

The BattleGrid icon should now appear in your Games group.  The 
program can be run by double clicking on the new icon.
