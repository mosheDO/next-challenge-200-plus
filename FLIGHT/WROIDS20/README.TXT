WINROIDS v2.00 README FILE - 26 May 1994


BRIEF DESCRIPTION

Winroids is a version of the popular game "Asteroids" for the Windows
environment.  It features 256 colour ray traced graphics, high speed 
action and full multimedia sound.


SYSTEM REQUIREMENTS

A fast 386 or 486 computer - 386/33 should probably be considered a
minimum, though the game is playable on a 386/25.

Windows 3.1 or higher.

A soundcard (optional).
