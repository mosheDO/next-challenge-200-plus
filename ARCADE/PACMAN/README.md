# Greedy (1996)

A Pac-Man clone by Eclipse. Navigate mazes, eat dots, evade ghosts, and collect power-ups.

Released in 1996 as shareware (v1.10). Also available at:
https://archive.org/details/msdos_Greedy_1996

## DOSBox Setup

- `machine=svga_s3` (in dosbox.conf)
- `memsize=63`
- Run `IAFIX.BAT` or `GREEDY.EXE` directly

The included `dosbox.conf` from Archive.org has the exact settings.
