
MAGIC CARPET 2 BASF DEMO
------------------------

	This is a five level demo of Magic Carpet 2, showing day, night and cavern type levels.

	If you wish to run the demo with sound and music, you must first run :-

		MC2SETUP

	This will invoke a program to allow you to configure the game for the sound card installed
	in your machine.

	To run the demo, type:-	
	
		CARPET2

	if you wish to run a particular level, type one of the following :-

		CARPET2 -LEVEL 0				(day level)
		CARPET2 -LEVEL 1				(night level)
		CARPET2 -LEVEL 2				(day level)
		CARPET2 -LEVEL 3				(night level)
		CARPET2 -LEVEL 4				(cavern level)


	If you wish to cheat, to show all spells:-

		Use the cheat - Press I then type 'RATTY'.
		ALT F1 - to access all spells.
		ALT F2 - (several times) more mana
		ALT F9 - (once) to gain free spell access, ie. without needing certain castle levels.
		Shift & C - Complete level.


IN GAME MOUSE CONTROLS
----------------------

    Mouse Motion Left/Right             Alter direction of flight left and right.       
    Mouse Motion Forwards/Backwards     Alter direction of flight up and down.
    Cursor Up/Down                      Increase/Decrease speed of flight.
    Cursor Left/Right                   Move sideways left or right.
    
    Left Mouse Button                   Fire left selected spell.
    Right Mouse Button                  Fire right selected spell.
    
    Shift & Left Mouse Button           Toggle through left marked spell list.
    Shift & Right Mouse Button          Toggle through right marked spell list.
    
    Control                             Bring up spell menu.
    Return                              Bring up map screen.
    Alt                                 In map screen, show player information.
    

SPELL SELECTION MENU MOUSE CONTROLS
-----------------------------------

    Left Mouse Button                   Select left selected spell.
    Right Mouse Button                  Select right selected spell.
    Shift & Left Mouse Button           Toggle on/off left marked spell.
    Shift & Right Mouse Button          Toggle on/off right marked spell.
    Left & Right Mouse Button           Fire spell without selecting


IN GAME KEY ACTIONS
-------------------

    P                                   Pause game.
    ESC                                 Quit game.
    SPACE                               Continue game once dead with castle, 
										end game once dead without.
    R                                   Change screen resolution.


IN GAME DETAIL OPTIONS
----------------------

    F1                                  Toggle sound on / off
    F2                                  Toggle music on / off
    F3                                  Toggle process speed
    F4                                  Toggle soften on / off
    Shift & F5                          Gamma correction
    F5                                  Toggle reflections on / off
    Shift F6                            Toggle panel transparency on / off
    F6                                  Toggle sky on / off
    F7                                  Toggle shadows on / off
    F8                                  Toggle map and icons on / off
    Shift & F10                         Toggle light sources on / off
    [                                   Increase screen size
    ]                                   Decrease screen size
