Jazz Jackrabbit Holiday Hare 1995 Edition 
       ** IMPORTANT INFORMATION **
            October 17, 1995

To install Jazz Jackrabbit Holiday Hare 1995,
switch to the directory containing the
JJXMAS95.EXE file and type JJXMAS95.

This version of Jazz Jackrabbit MUST BE
INSTALLED INTO A NEW DIRECTORY! DO NOT
install this version of the game into
your current Jazz Jackrabbit directory
or you will damage your old installation
of Jazz Jackrabbit and will then have to
reinstall your original game.
